# Asset Registry Added

We've established a new, centralized asset registry (`/.flexos/design/002-assets.md`) to effectively manage all digital assets including images, icons, videos, and fonts. This will streamline design workflows and ensure consistency across the project.

## Key Features

- **Categorization**: Assets are categorized into Images, Icons, Videos, and Fonts.
- **Detailed Metadata**: Each asset type includes specific fields for tracking, such as paths, dimensions, usage, and accessibility information.
- **Improved Organization**: Provides a single source of truth for all project assets.

This addition will help us maintain a well-organized and efficient design system.
